# UPPCL/UPPTCL Line Restoration Portal - Fixed

This Streamlit version fixes StreamlitDuplicateElementId by assigning unique keys to all widgets.

## Run locally
```bash
pip install -r requirements.txt
streamlit run uppcl.py --server.address 0.0.0.0 --server.port 8501
```

## Deploy on Streamlit Cloud
Upload/replace `uppcl.py` and `requirements.txt` in your GitHub repository, then reboot/redeploy the app from Streamlit Cloud.
